## 安装

```
[sudo] npm i mbo -g
```

## 使用

### 1.博客初始化

```
[sudo] mbo init [dir]
```

初始化博客文件结构

### 2.mbo new [postname]

新建文章

### 3.mbo serve [dir]

开启实时预览

### 4.mbo build [options] [dir]

生成静态HTML文件
