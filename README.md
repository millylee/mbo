# mbo

[![NPM version](https://img.shields.io/npm/v/mbo.svg?style=flat-square)](https://npmjs.org/package/mbo)

> 简单的 Markdown 博客生成

- *请使用 6.x + LTS 版本*

```
   __  __ ____   ____
  |  \/  |  _ \ / __ \
  | \  / | |_) | |  | |
  | |\/| |  _ <| |  | |
  | |  | | |_) | |__| |
  |_|  |_|____/ \____/
```

## 安装

```
[sudo] npm i mbo -g
```

## 使用

### 1.博客初始化

```
mbo init <dir></dir>
```

初始化博客文件结构

### 2.新建文章

```
mbo new <postname>
```

### 3.博客预览

```
mbo serve <dir>
```

### 4.生成静态文件

```
mbo build <options> <dir>
```
